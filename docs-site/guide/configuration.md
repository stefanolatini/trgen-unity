# Configurazione

## File di Configurazione JSON

TrGEN Unity supporta configurazioni complete tramite file JSON.

```json
{
  "networkSettings": {
    "ipAddress": "192.168.1.100",
    "port": 4000
  },
  "triggerPorts": [
    {
      "portNumber": 0,
      "pinType": "NeuroScan",
      "isEnabled": true
    }
  ]
}
```
